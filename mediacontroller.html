Ar7-MediaController

A custom media controller for those people who don't have media control in their keyboard
Made and Maintained by Ar7

Download-v2 Source Code

Note*: If you have the previous version installed then you have to delete it and install this release a small bug has been fixed for more info click here

How To Install The Media Controller

1. Download The App
2. Open Run WIN + R and copy paste this shell:common startup
3. Now Copy Paste the downloaded application into this folder




How To Use The Media Controller

1. By default the application will open on start-up
2. For Increasing volume WIN + F11 3. For Decreasing volume WIN + F10
4. For Muting WIN + F9
5. For Pausing the media WIN + F5
6. For Playing Next Track WIN + F8
7. For Playing Previous Track WIN + F7
