Ar7-MediaController

A custom media controller for those people who don't have media control in their keyboard
Made and Maintained by Ar7

Download Source Code

How To Install The Media Controller
1. Download The App
2. Open Run WIN + R and copy paste this shell:common startup
3. Now Copy Paste the downloaded application into this folder




How To Use The Media Controller
1. By default the application will open on start-up
2. For Increasing volume WIN + 0 and for decreasing WIN + 9 and WIN + 8 for mute
3. For Pausing the media SPACE + ALT + CTRL
4. For Playing Next Track ALT + CTRL + Left_Arrow_Key
5. For Playing Previous Track ALT + CTRL + Right_Arrow_Key
